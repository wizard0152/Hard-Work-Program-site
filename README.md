## Как открыть

## Как загрузить на GitHub Pages
1. Создай репозиторий на GitHub.
2. Загрузи `index.html`, `style.css`, `script.js`.
3. Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: main, folder: /root.
6. Сохрани и дождись ссылки.
